# LinkedIn Writer Agent — 사용 가이드

링크드인 글쓰기 AI 에이전트입니다.
**브랜드 설정 → Claude Code 열기 → 자연어로 요청** 세 단계로 끝납니다.

---

## 폴더 구조

```
Linkedin_writer_agent_Share/
├── .claude/
│   └── agents/
│       ├── linkedin-writer.md    ← 메인 에이전트 (전체 파이프라인 조율)
│       ├── brand-strategist.md  ← 브랜드 적합성 평가
│       ├── content-planner.md   ← 기획서 + 훅 3안 생성
│       ├── writer.md            ← 초안 작성
│       └── editor.md            ← 편집 + 품질 점수
├── brand_config.md              ← ★ 여기만 수정하면 됩니다
└── 사용가이드.md
```

---

## 설치 (딱 한 번만)

### Step 1. 이 폴더를 내 컴퓨터에 저장

원하는 위치에 `Linkedin_writer_agent_Share` 폴더 전체를 복사합니다.

---

### Step 2. brand_config.md 작성

`brand_config.md` 파일을 열어서 **내 정보로 채웁니다**.

```
내 브랜드 설정
├── 기본 정보 (이름, 직책, 회사)
├── 브랜드 포지셔닝 (어떤 사람으로 기억되고 싶은가)
├── 핵심 독자 (누구를 위한 글인가)
├── 콘텐츠 기둥 (반복적으로 다룰 주제)
├── 전문 배경 (신뢰도를 높이는 레퍼런스)
├── 글쓰기 톤
└── 하지 않을 것
```

> **이 파일만 수정하면 됩니다.** 나머지 에이전트 파일은 건드리지 않아도 됩니다.

---

### Step 3. Claude Code로 이 폴더 열기

```
Claude Code 실행 → 이 폴더 경로로 이동
```

터미널에서:
```bash
cd "폴더 경로/Linkedin_writer_agent_Share"
claude
```

또는 VS Code에서 이 폴더를 열고 Claude Code 실행.

---

## 사용 방법

Claude Code에서 아래처럼 말하면 됩니다.

### 기본 사용

```
링크드인 글 써줘. 주제는 [주제]야.
```

```
이 내용으로 링크드인 포스팅 만들어줘:
[쓰고 싶은 내용을 자유롭게 입력]
```

### 흐름 예시

```
1. 에이전트가 brand_config.md를 읽고 브랜드 파악
2. 주제가 내 브랜드에 맞는지 평가 (점수 제시)
3. 기획서 + 훅 A/B/C 3안 제시
4. "B안으로 해줘" → 선택 입력
5. 초안 작성
6. 편집 + 품질 점수 산출
7. 최종본 출력 (바로 복사해서 올릴 수 있는 형태)
```

### 피드백 후 수정

```
마지막 줄을 더 강하게 바꿔줘
훅을 A안으로 바꿔줘
전체 톤을 좀 더 가볍게 해줘
```

## 자주 묻는 질문

**Q. 에이전트가 "brand_config.md를 먼저 작성해달라"고 하면?**
→ brand_config.md 파일을 열어서 [이름을 입력하세요] 등 미작성 항목을 채우세요.

**Q. 글이 내 스타일과 다르게 나오면?**
→ brand_config.md의 "글쓰기 톤" 항목을 더 구체적으로 작성하세요.
→ 또는 실제 내가 쓴 글 예시를 brand_config.md에 추가하면 더 정확해집니다.

**Q. 특정 에이전트만 따로 쓸 수 있나?**
→ 가능합니다. 예: "writer 에이전트를 써서 이 기획서로 초안만 써줘"

---

## 팁

- **brand_config.md를 구체적으로 쓸수록** 글 퀄리티가 높아집니다.
- **내가 실제로 쓴 링크드인 글 2-3개를 brand_config.md에 붙여넣으면** 에이전트가 내 스타일을 훨씬 잘 모방합니다.
- **주제를 구체적으로 줄수록** 결과가 좋습니다. "AI 관련 글" 보다 "Claude Code로 비개발자가 에이전트 팀을 구축한 경험"처럼 구체적으로.
